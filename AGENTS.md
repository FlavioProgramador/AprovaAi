# 🤖 Diretrizes dos Agentes - AprovAI

Bem-vindo ao projeto AprovAI. Ao trabalhar neste repositório, você deve respeitar as diretrizes de desenvolvimento contidas nas nossas skills locais localizadas em `.agent/skills/`.

## Skills Disponíveis para Uso:

1. **`superpowers`**: Fluxo estrito de TDD, plano de implementação e micro-tarefas.
2. **`caveman`**: Modo de compressão e economia extrema de tokens de resposta.
3. **`frontend-design`**: Diretrizes de UI premium, HSL, responsividade e micro-animações.
4. **`overclock-mkt`**: Foco na conversão, copywriting persuasivo e design orientado a vendas.
5. **`deploy`**: Pipeline de deploy seguro (Azure, Docker, GitHub Actions).
6. **`backend-expert`**: Clean Architecture, API RESTful e desacoplamento.
7. **`auth-master`**: Criptografia de senhas, JWT seguro com HttpOnly e RBAC.
8. **`test-wizard`**: Cobertura de testes unitários, isolamento com mocks e testes E2E.
9. **`db-architect`**: Versionamento por migrações e índices de performance.
10. **`security-auditor`**: Higienização contra XSS e SQL Injection.
