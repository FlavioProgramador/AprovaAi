# 🛡️ Política de Segurança

## Versões Suportadas
| Versão | Suportada |
| ------- | ------------------ |
| 1.0.x   | ✅ Sim             |
| < 1.0   | ❌ Não             |

## Relatando uma Vulnerabilidade
Se você descobrir um problema de segurança no AprovaAi, por favor, não crie uma Issue pública. Em vez disso, envie um e-mail diretamente para `seu-email@dominio.com`. Todas as vulnerabilidades serão respondidas em até 48 horas.
