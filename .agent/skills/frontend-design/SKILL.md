# 🎨 Frontend Design Skill

Premium styling and rich user interface design guidelines. Focuses on premium look-and-feel, responsiveness, modern aesthetics, HSL color harmony, and micro-animations.

## 📐 Diretrizes de Design

1. **Aparência Premium**:
   - Evite layouts simples e cores genéricas (red, green, blue puros). Use paletas harmoniosas baseadas em HSL adaptados ou esquemas escuros modernos (slate, dark blue, metal-gray).
   - Implemente gradientes e efeitos de vidro (glassmorphism/backdrop-blur) para criar sensação de profundidade.

2. **Tipografia Moderna**:
   - Sempre utilize fontes elegantes do Google Fonts (Inter, Roboto, Outfit, Plus Jakarta Sans) em vez de fontes padrão do navegador.

3. **Micro-animações**:
   - Crie transições suaves para interações de hover, focos e estados de carregamento (spinners pulsares, fades, slides-up).

4. **Responsividade Estrita**:
   - Todos os layouts devem se adaptar a qualquer tamanho de tela (mobile-first ou layouts baseados em grids fluidos).
